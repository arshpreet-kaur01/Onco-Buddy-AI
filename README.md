# 🚀 Installation

Clone Repository

```bash
git clone https://github.com/arshpreet-kaur01/Onco-Buddy-AI.git
```

Go inside project

```bash
cd Onco-Buddy-AI
```

Install Packages

```bash
npm install
```

Create `.env`

```env
OPENROUTER_API_KEY=YOUR_API_KEY
```

Start Server

```bash
node server.js
```

Open Browser

```
http://localhost:3000
```
