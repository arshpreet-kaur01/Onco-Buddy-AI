# 🩺 Onco-Buddy AI

An AI-powered healthcare chatbot that provides general medical guidance, health tips, and educational information using OpenRouter AI.

## 🚀 Features

- 🤖 AI Chat Assistant
- 💬 Real-time conversation
- 🌐 Modern and responsive UI
- ⚡ Fast Node.js + Express backend
- 🔒 Secure API key using .env
- 🧠 Powered by OpenRouter AI

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- OpenRouter API
- Git & GitHub

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/arshpreet-kaur01/Onco-Buddy-AI.git
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```
OPENROUTER_API_KEY=your_api_key_here
```

Run the server

```bash
node server.js
```

Open your browser

```
http://localhost:3000
```
